──────────────────────────────────────────────────────────────
  SELETOR DE IDIOMA (PT / EN)
──────────────────────────────────────────────────────────────

  Todos os modelos possuem um seletor de idioma na barra
  de navegacao com dois botoes: PT | EN

  O idioma padrao e PORTUGUES.
  O usuario pode alternar para ingles com um clique.
  A escolha e salva automaticamente no navegador.

  Para adicionar ou alterar traducoes, encontre o objeto
  const translations = { pt: {...}, en: {...} } no
  JavaScript e atualize os valores para ambos os idiomas.

──────────────────────────────────────────────────────────────
  CORES
──────────────────────────────────────────────────────────────

  As cores principais sao definidas como variaveis CSS
  no topo do arquivo, dentro da secao :root { ... }.

  Para alterar a cor principal, encontre a variavel
  correspondente e insira sua cor no formato HEX.

  Exemplo:
    --terra: #C1440E;   substitua pela sua cor principal
    --gold:  #C9943A;   substitua pela sua cor secundaria

  FOTOS
──────────────────────────────────────────────────────────────

  Os modelos usam fotos de exemplo do Unsplash.
  Para usar suas proprias fotos:

       de exemplo no codigo HTML.

  OPCAO B: Fotos locais
    Coloque suas fotos na mesma pasta que o arquivo HTML
    e use: <img src="sua-foto.jpg" ...>
