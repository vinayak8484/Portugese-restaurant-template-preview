──────────────────────────────────────────────────────────────
  HOW TO GET STARTED
──────────────────────────────────────────────────────────────

  1. Open the HTML file in your browser.
     It works immediately with no installation needed.

  2. To edit the content, open the file in any text editor
     (Notepad, TextEdit, VS Code, Notepad++, etc.)

  3. Use CTRL+F (or CMD+F on Mac) to search for the text
     you want to change.

  4. Save the file and reload the browser to see changes.


──────────────────────────────────────────────────────────────
  THE 5 THINGS TO CHANGE FIRST
──────────────────────────────────────────────────────────────

  1. RESTAURANT NAME
     Search for the example name (Alma Cafe, NOBRE,
     BRASA Churrascaria) and replace it with yours. It appears
     in both the HTML and in the JavaScript translations
     object (search for the word translations).

  2. ADDRESS AND CONTACT DETAILS
     Find the example address and type in yours.
     Do the same with phone number and email.

  3. MENU AND PRICES
     The menu lives inside the translations object in
     the JavaScript. Search for "menu" and update dishes
     and prices for both PT and EN.

  4. OPENING HOURS
     Find the example hours and update them.

  5. GOOGLE MAPS
     Go to maps.google.com, search your address, click
     Share and then Embed a map. Copy the iframe code.
     In the HTML, find the iframe tag in the map section
     and replace it with yours.


──────────────────────────────────────────────────────────────
  LANGUAGE SWITCHER (PT / EN)
──────────────────────────────────────────────────────────────

  All templates have a language switcher in the navbar
  with two buttons: PT | EN

  The default language is PORTUGUESE.
  The user can switch to English with one click.
  The choice is saved in the browser.

  To add or change translations, find the object
  const translations = { pt: {...}, en: {...} } in the
  JavaScript and update the values for both languages.


──────────────────────────────────────────────────────────────
  DARK MODE
──────────────────────────────────────────────────────────────

  Every template has a night mode / day mode toggle button
  in the navbar. The user's preference is automatically
  saved in the browser.

  Alma Cafe and BRASA start in light mode.
  NOBRE Fine Dining starts in dark mode.


──────────────────────────────────────────────────────────────
  COLOURS
──────────────────────────────────────────────────────────────

  Main colours are defined as CSS variables at the top
  of the file, inside the :root { ... } section.

  To change the main colour, find the corresponding
  variable and enter your colour in HEX format.

  Example:
    --terra: #C1440E;   change to your main colour
    --gold:  #C9943A;   change to your secondary colour
